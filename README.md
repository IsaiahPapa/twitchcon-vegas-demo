## Running locally

You will need to use the environment variables. The following are needed.
```env
UPSTASH_URL=
UPSTASH_TOKEN=
NEXT_PUBLIC_TTS_API_URL=
```

Clone the repo, then run
```bash
npm install
npm run dev
```

Your app should now be running on [localhost:3000](http://localhost:3000/).
